Hole 01 — Overview

01 Game Overview

Hole 21 is a hands-on tabletop game that takes mini golf from simple to extreme. Players build their own course — from laying tracks flat on the floor to constructing ramps that go up, down, and everywhere in between. The track layouts are unlimited; your imagination is the only boundary.

Our balls are made from the highest quality materials, as are our putters. Start with a simple open course, then add obstacles round by round to crank up the difficulty. If a player completes the course within the time limit, they earn a bonus — speed and precision are both rewarded.

Hole 02 — Goal

02 Game Goal

The goal of the game is to complete all 21 holes using the fewest putts possible. The player with the lowest total putt count at the end wins.

✓ Win Condition
Complete all 21 holes in fewer putts than your opponent.
✗ Lose Condition
Finish with more putts than the opposing player.

Hole 04 — Mechanics

04 Game Mechanics & Logic

Main Actions:

  • Swing the putter and choose your course layout
  • Make the ball into the hole
  • Avoid obstacles placed on the track

Variables:

Variable NameStarting ValueWhat It Does
scoreTee Box (0)Tracks total putts taken across all 21 holes
health3How many obstacle hits a player can take per hole
timeRemaining60 secondsTime limit per hole — finish early for a bonus

If / Then Rules:

  • If the player makes a hole-in-one → add 10 bonus points
  • If the player hits an obstacle → lose 1 health
  • If health equals 0 → the game ends (lose)
  • If time equals 0 → the game ends (lose)
  • If the ball goes in the hole → round complete — advance to next hole
  • If player finishes under time limit → earn a speed bonus
Hole 05 — Game Loop

05 Game Loop

These steps repeat for every hole until all 21 are complete:

  1. Check for player input (key press or mouse click)
  2. Move the player and aim the putter
  3. Swing — check for ball collisions with obstacles or walls
  4. Update score (putt count) and health based on what was hit
  5. Check if ball is in the hole (win this round) or if health / time hit zero (lose)
  6. Player 1 finishes, then Player 2 goes — alternate until hole is complete
  7. Advance to next hole and repeat until Hole 21 is done
Hole 06 — Playtesting

06 Playtesting & Changes

Changes made after testing the game with real players:

⚠ Problem Found

Players said the early holes were too easy — no challenge getting the ball in.

✓ Changes Made

Added more obstacles on holes 5–21 and reduced the per-hole time limit from 80 to 60 seconds. Players also now choose obstacle placement before each round starts, adding strategy.

Hole 08 — Reflection

08 Designer Reflection

The hardest part was making the game balanced — especially deciding how many obstacles to put on each hole so it felt challenging but still fair. Playtesting was huge: we changed the timer and added more obstacles after real players tried it.

The game uses key programming concepts like variables (putt count, health, time), if/then rules (if ball hits obstacle → lose health), and loops (repeat for each of the 21 holes). Building this taught us how physical game design and coding logic connect in real ways.
